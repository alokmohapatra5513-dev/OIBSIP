# 🌐 Alok Mohapatra - Frontend Portfolio Website

Welcome to the repository for my personal professional portfolio website! This project serves as a modern, responsive showcase of my frontend web development capabilities, featured projects, and core technical skills.

---

## ✨ Features

- **Smooth UX Flow:** Integrated navigation system mapping dynamically across all sections (`Home`, `About`, `Skills`, `Projects`, `Contact`).
- **Interactive Componentry:** Features micro-animations including custom cursor tracking, animated progress skill bars, and hover transformations.
- **Responsive Architecture:** Fully optimized layout adapting seamlessly across desktop layouts, tablets, and smartphone displays.
- **Operational Interface Elements:** Structurally embedded contact forms, intuitive project visual grid arrays, and global social connect nodes.

---

## 🛠️ Technical Stack

- **Structure:** HTML5 Semantic Markup
- **Styling:** CSS3 Custom Properties (Flexbox/Grid layout frameworks)
- **Icons & Typography:** Remix Icons Engine & Google Fonts Integration (Poppins Spec)
- **Logic:** Vanilla JavaScript
