# 🌡️ Thermos. | Premium Temperature Conversion Web App

An elegant, single-page application engineered to process precision calculations across dynamic thermal measurement units (Celsius, Fahrenheit, and Kelvin). This web utility blends strict calculation metrics with a high-fidelity visual interface.

## 📊 Conversion Formulas Implemented

- **Celsius:**
  - $Fahrenheit = (C \times 9/5) + 32$
  - $Kelvin = C + 273.15$
- **Fahrenheit:**
  - $Celsius = (F - 32) \times 5/9$
  - $Kelvin = ((F - 32) \times 5/9) + 273.15$
- **Kelvin:**
  - $Celsius = K - 273.15$
  - $Fahrenheit = ((K - 273.15) \times 9/5) + 32$
